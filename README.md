# Research Digest Agent

## Overview
The Research Digest Agent is an intelligent system designed to synthesize and summarize academic research papers. It assists researchers, students, and anyone interested in keeping up with the latest research developments by providing concise summaries that highlight the purpose, methodology, and key findings of research.

## Features
- **Automated Summarization:** Automatically generates summaries of research papers.
- **User-Friendly Interface:** Simple and intuitive interface for easy navigation and use.
- **Multi-Format Support:** Supports various research paper formats including PDF, DOCX, and more.
- **Citation Assistance:** Provides citation formats for the summarized papers.

## Installation
1. Clone the repository: `git clone https://github.com/sumit-singh-99/Research-Digest-Agent.git`
2. Navigate to the directory: `cd Research-Digest-Agent`
3. Install the required dependencies: `npm install`

## Usage
Run the application using the command: `npm start`
